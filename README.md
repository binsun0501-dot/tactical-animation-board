# 战术动画板

面向团队运动的轻量 2D 战术动画白板。v1 先从足球切入，服务足球青训教练、校园足球老师、基础足球教练和业余球队教练/队长。

## 当前状态

已完成本地开发里程碑 0-4 的主要开发内容。当前处于 M4.1 iPad 现场白板交互修复中，等待用户继续 iPad 复测；不在本 README 中声明 v1 已完成。

已具备：

- 2D 足球场白板，本方棋子、对方棋子和足球。
- 棋子和足球拖动，跑动箭头和球路箭头绘制。
- Step 分步状态，新增步骤继承上一状态，每步独立位置、路线和说明。
- Step 播放、暂停、重播，展示模式和只读队员观看页骨架。
- localStorage 本地保存、最近保存读取、v1 JSON 导出 / 导入。
- 当前步骤 PNG 图片导出。
- 8 个足球动作示例模板，可查看、播放、复制为我的战术后继续编辑和保存；当前仅作为功能演示，真实训练前需按实际战术自行调整。

## v1 边界

v1 坚持轻量、2D、中文优先、手机/iPad 友好。第一版围绕足球现场白板、分步动画、保存/观看和示例模板闭环推进。

v1 不做：

- 3D；
- AI 自动生成战术；
- 视频识别或比赛分析；
- 复杂账号系统；
- 后端云同步；
- 大型社区；
- 模板市场；
- 篮球等多运动模式。

## 运行方式

安装依赖：

```bash
npm install
```

本地开发：

```bash
npm run dev -- --port 5173
```

构建检查：

```bash
npm run build
```
