面向团队运动的轻量 2D 战术动画白板。v1 先从足球切入，服务足球青训教练、校园足球老师、基础足球教练和业余球队教练/队长。

- 确认项目工作目录。
- 初始化 git 仓库。
- 配置 GitHub remote。
- 创建基础 README 和 `.gitignore`。

尚未进入阶段 1，未创建前端工程，未安装依赖，未开发 UI 或功能。

## v1 边界

v1 坚持轻量、2D、中文优先、手机/iPad 友好。第一版必须围绕足球现场白板、分步动画、保存/观看和基础模板库闭环推进。

v1 不做 3D、AI 自动生成战术、视频识别、复杂账号系统、大型社区、后端云同步作为必需能力，也不在第一版加入篮球等多运动模式。
