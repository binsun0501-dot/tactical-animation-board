# 战术动画板

面向团队运动的轻量 2D 战术动画白板。v1 先从足球切入，服务足球青训教练、校园足球老师、基础足球教练和业余球队教练/队长。

## 当前阶段

当前进入开发里程碑 1：基础编辑闭环。

已完成：

- 阶段 0：开发前准备。
- 阶段 1：最小 Web App 工程初始化。

里程碑 1 的目标是完成本地 Web App、足球场画布、棋子拖动、跑动/球路箭头，以及 Step 0/1/2 分步状态机制。

## v1 边界

v1 坚持轻量、2D、中文优先、手机/iPad 友好。第一版必须围绕足球现场白板、分步动画、保存/观看和基础模板库闭环推进。

v1 不做 3D、AI 自动生成战术、视频识别、复杂账号系统、大型社区、后端云同步作为必需能力，也不在第一版加入篮球等多运动模式。

## 运行方式

安装依赖：

```bash
npm install
```

本地开发：

```bash
npm run dev
```

构建检查：

```bash
npm run build
```
