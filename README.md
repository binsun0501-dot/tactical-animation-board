# 战术动画板

面向团队运动的轻量 2D 战术动画白板。v1 先从足球切入，服务足球青训教练、校园足球老师、基础足球教练和业余球队教练/队长。

## 当前状态

已完成并 push M0-M4.1，v1 candidate 总审已通过。当前进入 M5：公网 PWA 实验版交付。

当前版本是 `v0.1.0-alpha` 实验版，不是 v1 final，不是正式发布，也不代表真实教练反馈已经完成。

公网实验版地址：

https://binsun0501-dot.github.io/tactical-animation-board/

测试二维码：

![M5 公网实验版二维码](public/m5-test-qr.png)

已具备：

- 2D 足球场白板，本方棋子、对方棋子和足球。
- 棋子和足球拖动，跑动箭头和球路箭头绘制。
- Step 分步状态，新增步骤继承上一状态，每步独立位置、路线和说明。
- Step 播放、暂停、重播，展示模式和只读队员观看页骨架。
- localStorage 本地保存、最近保存读取、v1 JSON 导出 / 导入。
- 当前步骤 PNG 图片导出。
- 8 个足球动作示例模板，可查看、播放、复制为我的战术后继续编辑和保存；当前仅作为功能演示，真实训练前需按实际战术自行调整。
- 基础 PWA 配置，可用 iPad Safari 打开并尝试添加到主屏幕。

## iPad 打开方式

1. 用 iPad Safari 打开公网实验版地址。
2. 点击 Safari 的分享按钮。
3. 选择“添加到主屏幕”。
4. 回到 iPad 主屏幕，点击“战术动画板”图标打开。
5. 建议横屏使用。

说明：Safari 浏览器顶部栏会占用一部分空间；从主屏幕打开后，显示面积更接近正式现场使用。

## 实验版能做什么

- 摆放本方 / 对方棋子。
- 拖动棋子和足球。
- 画跑动线 / 球路线。
- 撤销路线。
- 分步骤记录和播放。
- 保存到本机。
- JSON 导入 / 导出。
- 当前步骤图片导出。
- 使用示例模板进行演示，并复制为我的战术后修改。

## 实验版不能做什么

- 不是正式版。
- 没有账号。
- 没有云同步。
- 没有多人协作。
- 没有后台数据库。
- 没有真实教练专业模板库。
- 没有 App Store / TestFlight。
- 没有 GIF / MP4 导出。
- 没有 AI 自动生成战术。
- 没有 3D。
- 没有篮球 / 多运动。

## 实验测试目标

本阶段重点只看三件事：

- 流畅；
- 简单操作；
- 简单理解。

## 反馈方式

M5 不做账号、表单后台或数据收集。试用后可把文字、截图、语音或现场感受直接发给项目联系人，优先反馈：

- 哪里卡；
- 哪里不会用；
- 哪个功能最有用；
- 哪个地方最不好用；
- 队员哪里看不懂；
- 下次训练或比赛是否还想继续用。

## v1 边界

v1 坚持轻量、2D、中文优先、手机/iPad 友好。第一版围绕足球现场白板、分步动画、保存/观看和示例模板闭环推进。

v1 不做：

- 3D；
- AI 自动生成战术；
- 视频识别或比赛分析；
- 复杂账号系统；
- 后端云同步；
- 大型社区；
- 模板市场；
- 篮球等多运动模式。

## 运行方式

安装依赖：

```bash
npm install
```

本地开发：

```bash
npm run dev -- --port 5173
```

构建检查：

```bash
npm run build
```
